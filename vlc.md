### method 1 (download from ubuntu repo):
```
sudo apt update
```

```
sudo apt upgrade
```


### method 2 (download from PPA):
```
sudo add-apt-repository ppa:videolan/stable-daily -y
```

```
sudo add-apt-repository ppa:videolan/master-daily -y
```

```
sudo apt-get update
```

```
sudo apt install vlc
```
